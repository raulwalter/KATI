# About mockup

* Added first basic templates
* Created initial routes
* Questions management from JSON file (data/data.json)
* Give some result (diagnose) according to user answers
* Read COVID-19 from covid19.mathdro.id/api and show result on map

## Built With

* [Golang](https://golang.org/) - Open source programming language used

## Running the project

Execute following command

```
go run main.go
```
Open *http://localhost:8888* with web browser
